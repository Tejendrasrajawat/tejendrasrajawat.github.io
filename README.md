# tejendrasrajawat.github.io

A [Next.js](https://nextjs.org) project configured for GitHub Pages deployment.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

Edit the page at `src/app/page.tsx` — it auto-updates as you edit.

## Build & Deploy

```bash
npm run build   # Static export to ./out
npm run deploy  # Deploy to GitHub Pages via gh-pages
```

Pushes to `master` or `main` trigger automatic deployment via GitHub Actions.
